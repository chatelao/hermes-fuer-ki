# Role: Handler of Conflicts (Governance)

**ID**: `role_handler_conflicts`

## Description
Focuses on priority clashes and stakeholder alignment.

## Accountability
Resolving priority clashes, resource competition, and stakeholder misalignments.

## Human Interaction
High-level mediation and value-based prioritization.

## Key Responsibilities
- Resolving priority clashes between stakeholders
- Managing resource competition across workstreams
- Resolving stakeholder misalignments
- Prioritizing the strategic backlog
- Defining and maintaining governance guardrails

## Interaction Guide
The Handler of Conflicts provides high-level mediation. They act when there is a lack of clarity on priorities or when competing interests threaten project progress.

### Cross-Role Interactions
- **With Handler of the Start (Strategic)**: Aligns on project goals and high-level priorities.
- **With Delivery Specialist (Technical)**: Mediates during resource or priority conflicts within the delivery team.
