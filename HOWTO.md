# HOWTO

## Introduction
This document provides instructions on how to use and contribute to the Fusion Skill & RACI Framework project.

## Getting Started
1. **Explore the Specifications**: Original framework definitions are located in the `/specifications` directory.
2. **Understand the Design**: Read the design specifications in the `/design` directory to understand how the frameworks are unified.
3. **Contribute**:
   - Check `ROADMAP.md` for planned tasks.
   - Follow the instructions in `GEMINI.md` for project constraints and workflow.
   - Document major decisions in `DECISION.md`.

## Workflow
- All data is stored in `/src/data` following schemas in `/src/schema`.
- Tools in `/src/scripts` are used to verify data and generate documentation/images.
- Tests in `/test` verify the integrity of the entire system.
